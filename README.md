
# Industrial Asset Performance Dashboard

## Project Overview
This Power BI dashboard analyzes industrial plant assets using a dummy dataset. It provides insights into machine status, purchase cost, department-wise distribution, manufacturer analysis, and location-wise asset tracking.

- Power BI
- Microsoft Excel
- DAX
- Power Query

## Key KPIs
- Total Machines
- Total Purchase Cost
- Running Machines
- Idle Machines
- Under Maintenance Machines
- Salvage Value
- Department-wise Asset Distribution
- Manufacturer-wise Machine Count

## Dataset
This project uses **dummy data** created for learning and portfolio purposes.
